  1. TC中 crtl+d -> b 快速跳转的我的hexo blog目录 d:/blog

  2. ->到TC的命令行， 相当于在 d:/blog 输入

指令，即相当于在打开当前目录。而vs code本来就是通过目录来组织，所以当前目录的git处理也被管理起来。
