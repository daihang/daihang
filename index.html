《空之境界》初感

发表于 2021-08-30 更新于 2021-12-26 分类于 随笔

一点反思

​ 距离看完这部动画也有快一个月，当时感概万千，就想写一个简单的初评，结果又一次验证了自己的拖延。现在看来拖延不是根因，而是这件事本身于我而言是有一定难度的：

  • 感触很多，但其实混乱
  • 一看就会，一写就废，其实自己写的能力其实很差

但正因为如此，更要勇敢去写：

  • 错过太多自己的灵感与美好。即使现在回头看自己的原来写的一些东西，虽然不见得文笔多好，但确实是自己的东西，自己至今看来依然觉得有意思。
  • 写吧写吧，说不定真就写好了呢？至少也能在工作总结上写得更好一点吧…

回到《空之境界》，确实时间过去较久，情节人物细节都不记得多少了，恐怕只能写写自己印象最深的几个镜头了，也许这并不一定是创作者本身想向观影者传达的重点，但既然自己记得，应该是自己最认同或者最在意的吧。

镜头一：初见两仪

一个暖光路灯的雪夜，男主黑桐打着伞走在回家的路上。这是条通往较高处的坡路，坡路边上还有保护车辆的护栏，一位和服少女伫立在护栏边静静地观看着华灯初上的城市，感觉就像是静谧雪景的组成部分。黑桐可能略奇怪少女没有撑伞，问候了一声：“你好”。少女自然的回头望向路对面的人，从漠然转为微微一笑…, 而黑桐也从一个自然打招呼的表情转为惊讶…

这应该是创作者刻意渲染的缘起。虽然为写文专门截了图，但实际上令我印象深刻的并不是少女的绝美面容，而是这个微笑的温暖和后续贯穿全剧场版两仪的清冷整体气质反差过于强烈，甚至印象中全剧再没有这个感觉的笑容。

嗯，确实是反差。本来雪夜是很冷的感觉，但近处清路灯和远处城市都散发着类似篝火的暖光。少女单薄的和服和漠然的回眸，与整体清冷的世界融为一体，但那响应男主问候的轻柔一笑，以一种对比强烈但又舒缓的节奏呼应了全剧整体的感受。

全剧就是给我的这样一种很难说清楚的感觉。黑桐作在全剧中的出镜率和存在感实在难以皮匹配男主的定位，而背负着血脉命运的两仪强大神秘，并在发展中确实获得了所谓的根性（贯穿死神之眼），能够看穿世界运行的规律并破坏一切，所以在各个其他人主动或被动的追求根性的故事中，两仪的参与始终有一种旁观，无所谓和认命的态度，仿佛这些故事都只是自己必然命运的预演(天地不仁，以万物为邹狗？)。

但恰恰由于男主的参与与纠葛，两仪在参与各个故事的过程一点点明晰了那微笑代表的的一点对温暖渴望的人性自我。男主可能并不真正明白两仪的全部背景和感知，但那个被初始一笑深深打动的他也在一个个故事里更加明确了自己对两仪的感情，不是占有，不是迷失自我，而且真的喜欢和你一起走下去的感觉…

唉…

实在写不下去其他有感触的镜头了，内心感受的想写出来真的好难。但确实感觉自己在意的东西稍微明晰了一些，就是喜欢这种不需要太多理由的相依的感觉…

慢热的特点

发表于 2017-01-17 更新于 2021-06-14 分类于 随笔

　　随着对自己了解的加深，再加上这段时间长跑，体检，踢球，和学习点新东西的结果和感受，感觉自己的特点是慢热。
　　想到的一个类比是：牛人相当于一部跑车，可以几秒内加速到超高的速度(进入状态快)，并且在可以在这个超高的速度能坚持比较长的时间(持续性好)，甚至爬升到极限速度（最佳状态）。而我的特点应该是加速比较慢，但如果慢慢起步，逐步进入状态，说不定还可以达到最佳状态并维持一段时间。而有时候越是急着加速，越是到不了最佳状态，还可能团团混乱。
　　两个典型的例子：

  • 踢球。如果一开始搞得很猛，很可能上气不接下气，很快体能耗完，并且整体表现一路下滑，2个小时的表现可能只有前10分钟比较高光；而如果一开始注意呼吸节奏，把动作幅度压倒身体状态比较舒适的程度（比如少过人，少上抢），等身体状态持续提升后，再开始尝试一些要求高的动作，可能就是前10分钟中规中矩，其余时段的表现都可以表现得中等偏上。
  • 工作。熬夜加班效率真的很低下，保证良好作息，工作的时候也不要急着一开始就处理很复杂的部分，慢慢持续稳定的向前走，把状态调整好后尝试攻坚的工作（其实攻坚的工作对我而言也不算多），长久持续稳定的输出，对我而言才是真正算是尽到全力。

20170115主v利物浦-又是一年双红会

发表于 2017-01-16 更新于 2021-06-14 分类于 文章

　　2017年第一场双红会，和以往都来有一些不太一样。话说小半个赛季过去，几乎完全不同的舆论。上次曼联是勉强保平，大家都说穆鸟已经过时，渣叔和瓜一样带来革新。现在专家们预测的则是曼联多半要赢。这人言啊…

半场小结

　　非常有趣的一场比赛，两支球队都把自己的风格打出来了。
　　这场比赛之前，如果有人说有一只队伍能够同时限制住卡里克，埃雷拉和博格巴的出球，我肯定是不信的，利物浦可以说是做到了… 当然，博格巴今天的心态有点问题。
　　想当年，曼联在英超横的时候，就是对上猛的队我比你细腻，对上细腻的队，我比你猛（是的，这里单指娜娜）。实际上这么干的基础是，对于猛队，我魔的猛也能做到针尖对麦芒。而在范厨子时代，我个人最痛心疾首的就是这种铁血精神的消失。今天，这种精气神回来了。面对疯狂逼抢的红军，我魔基本上做到了拼抢上不落下风，细腻上还能略胜一筹。当然远不如对阵其他队的时候来的流畅，毕竟，渣叔的利物浦真的是太勤勉了…

终场小结

　　几乎不能奢求更多了，太过瘾了。那一阵82：18的控球率的时候（不是范厨师的横传），相信不少人和我一样找回当年心悸的感觉了，你看小贝这不都紧张的快咬手指头了。
　　真是斗法啊！
　　我觉得穆鸟应该是这么想的：上半场抢的那么凶，多少该有点体力下降吧，我也上个能抢的（卡里克下，鲁尼上）狂抢你一下，反正今天卡神也被抢的有点失常。我上半场没你拼的凶，这会体力上拼刺刀我也该有点优势吧。 - 然后控球率就到82：18了，而且多是向前的传球，利物浦一度风声鹤唳…
　　我觉得渣叔应该这么想的：兄弟们，顶住，相信自己，我们一定守的住，然后该进攻的还得一如既往往上涌，只要逮住一个机会就能解决比赛了，毕竟我们一球领先啊。我靠，这危险的有点过头啊，前场完全拿不住球了还打什么反击，库蒂尼奥，小伙子，靠你了，前场稳住，曼联现在压上很大，你只要抓住一次机会就够了，我知道你最擅长干这个了！　－　然后就是库蒂尼奥好几个妙传，但队友直面门将的打门都打的很差（德赫亚威武，当然体力下降也是原因）。
　　再然后，就真的是拼刺刀了，一如当年的双红会。只是这个利物浦远比当年的利物浦来的华丽，现在曼联也算近几个赛季最好的时候了。
　　所以真的是太过瘾了。如果还有什么奢求的话，那就是库蒂尼奥重伤刚刚复出，马内又参加非洲杯去了，现在的利物浦的进攻还是要差不少的，这点从库蒂尼奥上场后巨大作用就可见一斑。真想看看满血的利物浦和曼联对上是个什么感觉。

vue.js初印象

发表于 2017-01-11 更新于 2021-06-14 分类于 随笔

最近比较有空，准备做一个spa来还几个技术学习债，其中最感兴趣的就是vue.js了。
记得是在之前项目中用underscore的时候，偶然看到vue.js，当时就觉得理念不错，但由于种种原因，当时的还是用underscore完成了任务。现在花了大概两天的时间把vue官网教程的基础部分看完，觉得学习过程相当平顺。特别对于使用underscore中碰到的一些不便和不得已自己拓展的地方，觉得在vue中都找到了更优雅而且更通用的解决方法，很有种茅塞顿开的感觉。
个人认为vue.js的文档写的尤其好，但是这种好绝对不仅仅是作者文笔好的原因，而是作者对vue.js要解决的问题有了深入的理解，并且对问题及其解决方案都做了很好的抽象，而且看上去代码优雅，据说性能还很好，这实在是难能可贵。
其中由问题到文档，问题到解决方案( 程序员的三种境界 )，也是自己一直在努力追求的境界. 必须要深入研究一下这个框架，一方面完善自己的前端技能，另一方面更重要,需要好好学习作者的思维方式和态度，争取有朝一日也做点有价值的东东出来。
