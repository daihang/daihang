合三为一的神游感

发表于 2016-12-28 更新于 2021-06-14 分类于 随笔

这事为了补已经拖了好久的几篇神游感 - 新增tag - 可怕的走神 ：

  1. 昨天晚上重看<<射雕英雄传>>。本来是陪33同学看会纸质书，居然发现看得很有些感触，一下子就看了四个小时，居然从头看到郭靖都遇到黄蓉了…
  2. 今天早上写完todo list。浏览新闻，看到权健预计投资10亿玩中超，一直不知道这家土豪公司到底是什么业务的，就去查了一下，由于其类似传销的经营模式，居然又多扒了近1个小时…
  3. 下午好不容易又工作状态不错，想着晚上回家后33同学睡觉后继续如何如何。结果在微信浏览了一篇关于中产如何保值资产的文章，和老婆讨论了近半个小时，然后就延伸到33同学读书问题，再然后就去查了武汉小学，初中和高中的各种信息，浏览信息过程中又看到其他过来人关于孩子教育的一些观点，被其中一篇讨论《小别离》和孩子教育问题的帖子吸引，又从中间跳到一个关于寒门不出贵子的天涯长文，居然一口气读完，一下子就4个小时又过去，而且到了凌晨两点了，又影响了正常的作息了…

其实当时好多感概已经记不清楚了，可怕的拖延症…

把记得的简单小结一下吧：

  1. 小时候看<<射雕>>的电视和小说，总是很期待几大高手的出场，特别是电视版中东邪带着面具的亮相，几乎是每每重看最期待的镜头之一。发现还是年纪变了，现在看小说，反倒对一些角色人物更有感概，比如说七怪，一诺千金，为义远走千里，即使知道郭小侠资质平庸，也义无反顾，尽人力而听天命。比如马钰，为人低调，恬淡守静，但也会远赴大漠，秘授郭小侠心法，只为弥补师弟因为争强斗狠引起的无谓纷争，并不在意要是输了后门派丢脸的小节。还有大漠英雄，金戈铁马，也是各有性情，情谊，责任，野心和抱负，看上去简单，却也生动直接，比后来写实的多的《鹿鼎记》来的痛快许多；

  2. 当然，感概虽多，但最喜欢的还是七公的这句话：

    “老叫化一生杀过二百三十一人，这二百三十一人个个都是恶徒，若非污吏、土豪恶霸，就是大奸巨恶、负义薄幸之辈。老叫化贪饮贪食，可是生平从来没杀过一个好人。裘千仞，你是第二百三十二人！”

  3. 看权健。 一是感概这种插边球的传销居然可以产生这么强的资本能力，社会真的浮躁如斯？二是感概传销还可以这么高调，莫非足球真是道路正确游戏？
    但是反思这种传销模式的力量，一方面当然是大家都想快速致富的急躁心理；但另一方面，这个世界的快速发展，留给许多不是热门行业，甚至是淘汰行业的从业人员（尤其是年纪偏大的）的选择又有多少。又想起罗胖子说的那个老铁匠该怎么转型的问题…

  4. 寒门不出贵子这篇文章的逻辑，刚开始看觉得很有道理，后来即使觉得有点以偏概全，但确实也说明了这个社会（甚至是整个世界）的一些普遍现象。如果贵子说的社会地位，郭靖算寒门么，后来算贵子么？马云算寒门么，现在应该算是贵子了？七怪算寒门么，后来算贵子么？也许幼稚，但我还是觉得关键的问题是贵子的定义，这方面我真的还想不太明白，但我觉得最关键的应该还是人应该找到自己的立足点，最好是有兴趣的，至少是可以自我安慰的。
    所以，关于小孩教育，最关键的，依然是：
    （1） 做好自己，没有什么比言传身教更好的教育模式。希望孩子以后过得好，就要自己现在过得好。
    （2） 社会是很有问题，但这些问题并不是只在小孩的教育上。我们作为成人本来也在这光陆怪离的世界中修炼成长，所以参考第一条，首先我们要积极应对自己生活中的各种问题，而这种应对的方式，一定会在我们孩子身上得到传承。也就是，当你觉得你自己过得好的时候，你也会认为你的孩子不会过得差的。
    （3）由2联想到《西部世界》,突然感觉有些不同的收获：

    在那无序的混乱中，我选择看到美好。

雾霾感觉比较靠谱的分析

发表于 2016-12-20 更新于 2021-06-14 分类于 随笔

这是补的2016-12-10的一篇神游感，当时应该挺忙的，不知道怎么就走神去看了一篇很长的雾霾分析的文章。个人判断此篇分析应该是比较靠谱的，附一下文章的链接:
《雾霾的真相》——一个环保部门公务员的稽首自白

作者还是煞费苦心了的，为了让一些没有耐心的人也能有个大概印象，一开始就把小结放在文章的开头：

雾霾肆虐的主要原因，并不是我们常常听说的那些因素，而是因为环境监管体制存在各式各样的漏洞，企业节省成本，燃煤、工业排放、机动车等废气污染治理设施停止运行（尤其是夜间的偷排、直排！！！），导致大气污染物实际排放量与理论排放量比较，成十倍几十倍的增加，大大超过了环境的自我净化和扩散能力。

看完文章后个人的一点收获：

  1. 作者说明主要原因来自企业特别是民企的偷排，政府的重要责任是监管不力，所以文章在天涯上并没有被封；
  2. 凡事看两面。民企确实起到了解决结业，搞活经济的重要作用，但是其天生的逐利性如果没有合适的监管确实会导致这种没有良心的行为；
  3. 感觉又绕回了理想主义的老路，如果没有权利寻租，监管是不是就有力了；
  4. 既然没有理想主义，像这种事真的存在多方共赢的方法么？唉，理工生想不清楚。

程序员的三种境界

发表于 2016-11-27 更新于 2021-06-14 分类于 随笔

最近偶有所感，觉得程序员入门后有三种境界：

  1. 能解决问题。不说高大上的，普通程序员的大部分工作完成某个任务，或者解决问题，很多人其实都不知道自己要解决的问题是什么，或者知道了也不着怎么解决问题，而一个合格的程序员应该拥有良好的沟通能力和技能基础，加上不错的google技巧(是的，google)，就应该解决大部分日常工作中遇到的问题了。

  1. 能解决问题并总结为文档。总结成文档，并不是局限于单纯的记录，而是对遇到问题和解决过程的回溯和思考，这一般方面有利与以后更快的解决同类问题，而且可以分享给他人，降低沟通成本。

  2. 能解决问题并总结为代码。总结成文档，以后在参考的时候，其实还是有个再学习的过程，而对于程序员来说，实际的成果一般都是代码，特别是云时代，环境运维也都在往代码自动化方向发展的今天，代码的作用越来越核心。如果把遇到问题，实现成可重用的代码，那说明一是对问题的通用性和特殊性，已经理解的很透彻了，更重要的是，通过代码直接提供了可运行的解决方案。

这里设想一个场景来解释一下：某项目已经存在很久且功能一直够用，但是并没有使用流行的各种框架，而是通过一个自建的简单MVC框架实现了类似的功能。现在这个项目运行的不错，但要增加一些新功能，而且招入了程序员A, B进行开发。

A一看项目：我靠，这是什么玩意，这用的什么时代的东西，好多代码都不知道干啥的，管他三七二十一，反正我要做的那么点功能，我就在debug的过程中，随便加一下能跑过去就行，可能hardcode, 可能自建一些变量而原来的代码本来有一些共用的变量，还有可能本来应该写在model层的代码，就写到control层了… 好吧，其实这样的人真的不能算入门…

B浏览了一下代码的整体结构，基于自己MVC的基础，对现有代码的逻辑结构有了大致的理解，对于现有框架已有的功能能够充分利用，对于新加的功能，如果现有框架不支持，能够通google查找已有解决方案，并将别人的方案结合自己项目的情况，合理的引入并解决问题。

当前release完结后，B一边维护这个项目，同时也在做其他项目的开发，隔了一段时间，这个项目又有新需求，招入新人C做，并委托B帮指导。结果B发现一段时间不做之后，自己本身基本上忘光了，捡起来花了不少时间，而且在指导C的过程中，同样的问题可能会说好几遍。于是B痛下决心写了份文档，在写文档的思考和下笔的过程重，B发现自己该项目的理解更深了一层，而且甚至觉的其中有些通用的东西可以用在自己其他的工作中。另外，以后有其他D, E, F…问相关内容的时候，B可以直接先把文档给对方，然后只用对文档中别人不理解的地方做出解释就可以了。

慢慢的B又发现，即使有文档，但每个人基础背景的不一样，对文档的理解或多或少有一些偏差，还是带来实际搭环境或者编码过程中的一些问题。随着B的经验的提升，他发现其实有80%的工作是可以自动化，而剩下的的代码也有好多模式化的东西，于是B参考一些先进的技术，为项目写了自动建立开发环境的代码(比如vagrant或者docker)，另外开发了一个简易的命令行工具，可以生成一些代码模板，这样，对于项目的新人可以通过一个命令行就建立起可运行的开发环境，从而更容易的理解项目；而对于新人或者有一定经验的开发人员，可以通过命令行工具，生成一致的模板代码(model, control之类)，一方面提高了开发效率，一方面也更好的保证基础代码结构的一致性。而B的技能也得到了提升和应用，个人成就感得到满足，代码也更有价值。

其实，个人觉得，这个真的不一定要等到遇到问题才去写文档，写工具。应该作为一个习惯，对于每天的工作，首先至少总结记一下文档，在这个基础上思考一下是否能够代码化，并合理规划实现，才是一个程序员的自我修养。

20161119 vs 阿森纳后阶段性小结

发表于 2016-11-20 更新于 2021-06-14 分类于 文章

　　借这场比赛，也说说前两场和小结下曼联当前的状态：其实从赫雷拉和马塔的主力位置确认以来，曼联的情况其实一直是在好转的，伯恩利那场如果不是对方门将的神勇，就该一场大胜的。斯旺西这场果然就是个干净利落的3：1。

　　个人觉得，这两场充分说明了穆鸟中后场调试已经趋于稳定了。其中阵型应该就是4231和433的动态切换。而最不会随便动的三个人，应该就是博格巴+赫雷拉+马塔了。其中，博格巴和赫雷拉决定了攻守平衡，而马塔决定了进攻的前场串联。

  • 4231的话，应该是主场面对相对弱队的阵型，特别是后防人员齐整（比如说baily在），防守比较有信心的时候。
  • 而433的话，则会在面对强队的时候，增加卡神作为拖后组织核心上来协助防守。

　　这两个阵型，貌似穆鸟一直在用，而现在才渐入佳境的原因，个人觉得最关键的应该就是找到了赫雷拉。多次试验之后证明，赫雷拉才是博格巴最适合的中场搭档，因为：

  1. 积极的拼抢。这是基础，当然这点费莱尼和施耐德林也会做得不错。
  2. 优秀的技术功底(停球，护球，传球)。这个异常重要，可以大幅减少中场失误，可以帮助后防出球，可以和博格巴和马塔倒起来，还可以作为博格巴之外的中场第二出球点。而这也是也是费莱尼和施耐德林不足的地方。
  3. 良好的位置感。在博格巴冲上去后，可以留在后腰或者中场的位置，对对手的反击进行拦截和延缓。

　　可以这么说，博格巴和赫雷拉的搭档，保障了曼联的攻守平衡，起码是中后场不会随便被对手冲的乱其八糟了。这应该是曼联可以坐7望4保证。

　　而另外一个中后场的妙笔，就是卡神的启用。拜利受伤后，我一度以为后防会启用斯莫林+布林德的这一上赛季相当成功的组合。布林德虽然身体力量没有优势，但是预判一定程度上可以弥补，而他的出球却是曼联后方特别缺乏的，凑合几场应该还是可以的。但是穆鸟显然期待更稳妥的方案，牺牲一点进攻，增加卡神打拖后组织核心来稳定后防，和协助后卫线出球。这对曼联后防来说太重要了，因为曼联的后方并不缺乏强度和硬度，反倒是逼抢下的慌乱和失误极为致命，而卡神的协助，意味着即使面对较强的对手的时候，后防依然是可信赖的。一个真正稳定的后防，是曼联进入前4的最大保障(想想爵爷的后期，和范厨师的第一个赛季).

　　有了稳定的中后场，剩下的就看怎么进攻了。遗憾的是，曼联的进攻现在只能说是打到哪算哪，没有明确的套路:

  • 上伊布，有前场支点(这点在对手增加对伊布的针对性防守后，效果有所下降)，有和博格巴在进步的配合，有禁区的震慑力。但是没有了速度，大范围扯动，和高速的传切，所以密集防守破坏能力不足。但这不完全是伊布的问题，如果其他人和伊布在配合的默契和精巧能够更上一个台阶，就不好说了…
  • 鲁尼可以说是好一场坏一场。最关键是鲁尼基本不再有纵向突破能力，现在其他队对鲁尼已经没有畏惧感。所以状态不好的时候，基本功(短距离的一脚传球，护球，停球)的略差，导致在前场密集防守的狭小的空间中，失误频频。鲁尼即使不行了，也不该衰到这个地步。个人感觉，几场替补，和对手开始有点轻视他后，最近2-3场鲁尼的状态开始有所回升，基本可以说球不会在他脚下丢掉，跑位和传球质量都颇高。我觉得这可能是最好的结果，不再承担过重的责任，慢慢爬升到巅峰期的8成功力，就可以了。
  • 马夏尔看来是场外影响不小，基本功身体都还在，就是踢得不坚决，没灵性了，唉，人都是还是要为自己的错误买单，希望早点走出心理问题吧。
  • 拉斯福德倒是一直进步，但是还不够，尤其是433的时候，单兵作战能力不够强，其最大的优点还是在禁区前沿和队友的配合，和禁区内的冷静果断，所以4231的时候他应该发挥的更好。当然，这些随着其的飞速进步，还有很大的想象空间。
  • 马塔当下真的太关键了。基本的进攻手段，在队间差距日渐缩小的英超，其实都差不多，这个时候强队的在进攻上比较突出的杀手锏，就显得尤为关键了。切尔西有哈扎尔，阿森纳有374+272,曼城有TMD瓜迪奥拉，利物浦和托特拉姆有高压快攻，而曼联的进攻目前基本靠个人发挥… ，所幸让这种个人发挥真正有所不同的就是马塔了，极狭小空间的接球和带球，左右两边的内切，小范围二三人配合，不惜体力的跑位接应，一定程度上保证曼联进攻的连续性和控制力，而这个进攻连续性和控制力，就是强队的标志之一了。
  • 在中后场稳定，和mata的进攻核心地位明确后，林嘉德，姆希塔良和德佩等是不是也该多获得一些机会，而且是不是也得开始调试一下轮换阵容了…

　　所以，如果进攻和轮换的问题也解决好了，夺冠还是可以期待一下的。

　　好了，可以回到阿森纳这场了。这场可以说是433阵型的完美一战，可以说中后场组织的相当有序，阿森纳其实真的没得到过太好的机会。另外看到一点不同的是，在伊布鲁尼都不在的时候，前场的速度一度打出来了，虽然配合还谈不上特别流畅和精妙，但如上所说，曼联的进攻还有很大的提升空间，调校的好，就是面对不同的对手可以用不同的组合打出都有针对性的进攻，但如果不好，则会是没有明确进攻，混一个算一个的节奏了。穆鸟一直不算进攻大师，但只精练几个套路，相信应该还是做得到的。

　　另外这场继续暴露一个防守上还需要注意的问题，就是头球的防守。斯旺西和这场防定位球头球的时候，站位盯人都一些问题。这个通过微调和强调应该较快得到解决。

　　不管怎么说，个人觉得曼联确实是在往正轨的方向走，后面的比赛应该是越来越值得期待，哈哈！

　　另外，尼玛的大吉鲁，能不能不要选跟曼联比赛的时候打脸… …
